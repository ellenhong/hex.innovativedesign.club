# HEX: (H)one & (Ex)plore with Innovative Design

HEX Workshops are a workshop series hosted every semester by Innovative Design.
Each workshop is taught by an InnoD member or alumni and free for all Cal
students.

### Installation
```bash
$ npm install -g gulp
$ npm install
```

### Development
```bash
$ gulp
```
